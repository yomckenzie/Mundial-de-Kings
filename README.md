# Kings World Cup - ChessKing

Aplicación web de pronósticos para el Mundial de Kings. Los usuarios registran sus predicciones de resultados de partidos, acumulan puntos por aciertos y canjean premios.

## Scripts disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Compila para producción
- `npm run preview` - Previsualiza la compilación de producción
- `npm run lint` - Ejecuta el linter

## Tecnologías

- React 18 + Vite
- Tailwind CSS + shadcn/ui
- Almacenamiento local (localStorage)
- React Router v6
- @tanstack/react-query
- recharts (gráficos)
